# 🤖 Trading OS - بوت التداول الذكي

بوت تداول آلي متكامل يعمل على تليجرام مع واجهة Mini App، متصل بمنصة BingX للتداول الفوري.

## ✨ المميزات

- ✅ تداول آلي 24/7
- ✅ تحليل فني متقدم (RSI, EMA)
- ✅ إشارات شراء/بيع لحظية
- ✅ إدارة مخاطر ذكية
- ✅ دعم +500 عملة رقمية
- ✅ واجهة عربية سهلة
- ✅ قائمة مفضلة للعملات النشطة
- ✅ إشعارات فورية بالصفقات

## 🚀 طريقة التثبيت

### 1. متطلبات التشغيل
- Node.js v16+
- MongoDB
- حساب BingX
- بوت تليجرام من BotFather
